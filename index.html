FlowAgility streaming demo

To start:

  • Make sure your browser is set to 1280px x 720px. The layout is hardcoded to these specs.
  • Add your background image to `/img/perroenpista.png` and should be 1280px x 720px
  • Press the S key to start the video overlay (or click here)
  • Press the R key to show the results - this will show Combined results if available, otherwise Best results
  • Press the N key at any time to bring in the latest data from the server
  • Press the H key to hide all UI elements
  • Press the back key or left arrow key to show this screen
